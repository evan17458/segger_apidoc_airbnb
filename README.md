# Airbnb 訂房 API 文件

### 先決條件

**Node 版本 14.x**

### Cloning the repository

````shell
git clone https://github.com/evan17458/airbnb-clone

### 安裝包

```shell
npm i
````

### Setup .env file

### 設定 .env 檔

```shell
npx prisma db push

```

### 啟動應用程式

```shell
npm run dev
```

## 可用指令

使用 npm 運行命令 `npm run [command]`

| 命令  | 描述                                     |
| :---- | :--------------------------------------- |
| `dev` | Starts a development instance of the app |
